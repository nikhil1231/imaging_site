VISION & IMAGING IMPACT

How we're changing the world.

Automatic Inspection

Factories are increasingly relying less on man-power and more on robotic and automated systems. With factory robots that can 'see', defect recognition and quality control processes are greatly improved.

Read More

Artificial Intelligence

Artifically Intelligent computers can, through machine learning, use visual data received from pictures or videos for example. Different and exciting data can be extracted from a picture or video if we analyse it using Vision and Imaging analysis. This allows more diverse and nuanced applications of AI possible.

Read More

Image Organisation

This is automatically categorising photos based on what is in the photo. This, for example, can allow a user to search through a library of photos using keywords; photos including objects or features that relate to that word can be given without any need to manually tag the photo previously.

Read More

Vehicles and Driving

'Seeing' cars, planes, drones and amphibius vehicles have many exciting possible uses. Vision in these vehicles can enhance autonomous vehicles, driving safety, navigation, surveillance and military applications (for example, guided missiles).

Read More

Medical Image Processing

Image inspection and analysis can help doctors analyse patient images from all kinds of scanning methods such as endoscopy, MRI and ultrasound scans, by identifying and enhancing certain features in an image.

Read More

Visual Effects

Visual Effects in film and animation have progressed impressively over the past few decades thanks to the financial backing of the film industry. Vision and Imaging technologies can be used to enhance visual effects in both the filming and editing of a movie.
